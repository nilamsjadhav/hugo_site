Providing Options in Request (Fetch API)

October 13, 2022

  • fetch() method takes init object as second parameter that allow us to control a number of different settings.

  • We can specify following options in request.

    • method
    • headers
    • body
    • mode
    • credentials
    • omit
    • same-origin
    • include
    • cache
    • referrer
    • referrerPolicy
    • integrity
    • keepalive
    • signal

  • For example,

        fetch('https://pokeapi.co/api/v2/pokemon/1', {
        method: 'GET',
        headers: { 'Content-Type': 'application/json' },
        mode: 'no-cors',
        cache: 'no-cache',
        credentials: 'same-origin' })
    .then(data => data.json())
    .then(pokemons => console.log(pokemons));
